    # Attributes

    属性名 类型 默认值 说明 补充说明
    id Number 0 用于树状的高亮回显 此处应使用 v2 属性语法糖.sync 配合使用
    data Array [] 树状数据格式(具体可参见 elementui 的树形结构) 如果要使节点高亮回显应在数据中加入 id 字段
    v-model string/number - input 框中的回显值字段 -
    defaultProps Object - 详见 elemenui 中 tree 组件 prop 属性说明 -

    说明 ： 该组件暂时只支持单选模式 功能暂不完善 后续会根据业务进行补充

    # Events

    事件名称 说明 参数
    handleGetvalue 节点被点击时回调 {data:节点对应的原始数据；node:节点状态数据(具体数据参考 elementui)}
